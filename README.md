# Coffee Personality Quiz

A fun personality quiz that matches you with your perfect coffee based on your lifestyle and preferences. Answer 7 questions and discover which brew suits your personality.

**Live Demo:** [quiz-project-jade.vercel.app](https://quiz-project-jade.vercel.app)

## Features

- 7 personality-based questions
- 4 coffee personality types: Cozy Classic, Sweet Enthusiast, Health Nut, Indulgent Treat
- Percentage breakdown of all personality matches
- Confetti celebration on results
- Mobile-responsive design

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
